# What does this project do?
This project is a Python program that takes existing list, in this case the list of fruits, and puts each item in alphabetical order.

# How to Run Project
1. Open Python file
2. Download or clone this repository
3. Open project folder in a code editor or terminal
4. Run the file using the command "python abc_sorting.py"
5. Output can be viewed in terminal

# What I learned
- How to create a list
- How to store a list in a variable
- How to use Pythons built-in methods, specifically .sort(), which rearranges each string alphabetically
- How to create a loop that repeats each element in a list
- How to tell Python where to get the elements for the loop

# Requirements
- Python 3.14 or later
